TRUSTEE APP — DEPLOYMENT CHECKLIST
==================================

1. Confirm app.py and database/db.py compile
2. Confirm local routes load successfully
3. Confirm uploads folder path is writable
4. Confirm requirements.txt is present
5. Confirm wsgi.py is present
6. Confirm environment variables are set:
   - APP_ENV=production
   - FLASK_DEBUG=0
   - SECRET_KEY=<real secret>
   - PORT=<host port if applicable>
7. Confirm SQLite file path strategy for deployment
8. Confirm generated reports and upload folders are writable
9. Confirm `trustee_app.db` path is stable and writable in the deployment environment
10. Confirm uploads directory is stable and writable in the deployment environment
11. Choose deployment target:
   - local server
   - Render
   - Railway
   - VPS / Ubuntu + Gunicorn + Nginx
12. Use a production WSGI server instead of Flask debug server
13. Confirm HTTPS / secure cookies in production
14. Confirm direct export endpoints remain protected by ROLE_RULES
15. Confirm media file serving only resolves files under uploads/
