Guide

Use this page to understand what the platform does, where to begin, and how to manage users and roles.

What this app is

This is a trust operating platform for learning, planning, documenting, reporting, and reviewing trust-related workflows.

Start Here

  1. Login
  2. Review your dashboard
  3. Use the top navigation to move between modules
  4. Admins: create users first
  5. Trustees: begin with Portfolio or Workflow
  6. Viewers: begin with Portfolio and the shared learning modules

Admin Setup Guide

  1. Go to Admin → Users
  2. Click Create User
  3. Enter username and password
  4. Select a role:
    • Admin — full system access
    • Trustee — operational access
    • Viewer — read-only access
  5. Save the user
  6. Use Reset Password to change or assign credentials later

Role Overview

  • Admin: full system access including users, roles, permissions, security, exports, and oversight views
  • Trustee: operational trust management including workflow, portfolio, fiduciaries, genealogy, media, and reports
  • Viewer: read-safe access to portfolio and shared educational / platform modules

How to Trace Your Genealogy

  1. Start with the closest known relative and work backward one generation at a time.
  2. Record full names, dates, places, and family relationships before drawing conclusions.
  3. Archive each source clearly by platform, title, and reference or URL.
  4. Use the trace summary field to describe what the record appears to prove.
  5. Use the guidance prompt field to write what should be checked next.
  6. Do not rely on a single record when proving a parent-child or lineage connection.
  7. Look for a second confirming source such as a census, obituary, deed, church record, or family record.
  8. Mark the verification status honestly:
    • unverified — only a lead
    • in review — under active tracing
    • partially verified — some support, more needed
    • verified — supported by sufficient confirming evidence

Main Modules

  • Learning Hub: trust types, forms, and educational content
  • Videos: supporting how-to media
  • Workspace: planning sandbox
  • Discussions: structured discussion threads
  • Decision Engine: guided suggestions
  • Execution Engine: tasks and workflow
  • Documents: template-based draft generation
  • Visualization: oversight and timeline view
  • Reports: PDF/report generation

Where to Begin by Role

  • Trustee: Home, Workflow, Portfolio, Fiduciaries, Reports
  • Viewer: Portfolio, Learning, Videos, Workspace, Guide
