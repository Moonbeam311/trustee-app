Simulation vs Real Execution Guidance

Use Simulation to practice the workflow, test packet logic, or train a fiduciary user without treating the transfer as an executed real-world action.

Use Real only when the trustee is intentionally documenting an actual transfer, funding action, asset movement, account change, or trust administration event.

Simulation records are not executed transfers. Real execution should be supported by the trust record, packet readiness review, trustee authority, and supporting documents.
