Personal Property Support Document Categories

  • Item description clarification note
  • Physical possession transfer note
  • Condition and identification memo
  • Custody/location record
  • Supporting note for serial number, markings, or distinguishing traits
  • Optional delivery/receipt support note
  • Property-specific packet cover note

When Personal Property Support Docs May Help

  1. When the asset is tangible and identification depends on physical description.
  2. When possession, custody, or location is part of the control story.
  3. When multiple similar items could create ambiguity without extra description.
  4. When a later reviewer may need more than the base packet to identify the property clearly.
  5. When delivery, custody, or condition should be documented more carefully.
