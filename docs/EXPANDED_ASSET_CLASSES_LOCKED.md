# Expanded Asset Classes Locked

## Confirmed Principle
The wealth system is expanding its asset model. It is not narrowing assets into a single generic 'All Assets' concept.

## Confirmed Asset Classes
- real_property
- tangible_property
- financial_property
- business_interest
- intellectual_property
- digital_property
- contractual_right
- ucc_secured_interest
- other

## Important Clarification
The 'All Assets' dashboard view is only a reset/default view. It must not replace the broader asset classification model.

## Examples
- STEM Nation -> intellectual_property and potentially digital_property
- recipes -> intellectual_property
- websites/domains/hosting -> digital_property
- branding/code/curriculum/media -> intellectual_property
- truck/trailer/equipment -> tangible_property
- land/buildings/lots -> real_property
- UCC filings and secured commercial rights -> ucc_secured_interest

## Product Meaning
This supports the broader direction of the system as a trustee-first and fiduciary-first wealth management operating system rather than a narrow real-estate or form-driven application.

## Next Build Action
Update asset-entry forms and asset dashboard filtering so all locked classes appear clearly and remain available in the UI.
